## How to run

1. Clone this repository
2. Install dependencies
```bash
npm i
```
3. Run the front-end with
```bash
npm start
```
4. Finally access http://localhost:3000 on your favorite browser 
- Suggested API: https://mock-api.bootcamp.respondeai.com.br/api/v2/linkr
